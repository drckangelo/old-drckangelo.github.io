Grocey, grocey-delivery app

Description

  • Derick David

  • 1st March, 2017

An app that lets you order groceries from home with one tap of a button. It will then be arrived to your doorstep ASAP and efficiently. Features: Click-to-order, Door-to-door, delivery Quick-reorder, Order status tracking, Map-based search, Updated grocery goods

Technology

  • Swift for iOS Development
  • PHP and mySQL for Backend
  • Watson API
  • Cocoa Touch Frameworks
